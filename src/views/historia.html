Caracteristicas

Unix

Linux se basa en sistema Unix de los años 1970 y 1980 Un OS esrito en Lenguaje Ensamblador usados en los microprocesadores.
Luego reescrito en C.
La Empresa AT&T no podía comercializar Unix al principio, debido a un caso legal quelo prohibia, entonces se lo dio a personas o instituciones que lo pedían "en negro".
Luego se liberó de la limitación legal y empezaron a comercializarlo como un producto que no podia ser modificado por los usuarios.
Un sistema de este tipo utiliza un kernel monolítico, el kernel de Linux, que se encarga del control de procesos, redes, acceso a periféricos y sistemas de archivos.
Los controladores de dispositivos se integran directamente con el kernel o se añaden como módulos que se cargan mientras el sistema está en funcionamiento.

GNU

Década de 1980: Richard Stallman, un programador y defensor del software libre, fundó el proyecto GNU en 1983. Su objetivo era crear un sistema operativo completamente libre y compatible con Unix. Filosofía del Software Libre: Stallman abogó por la libertad de los usuarios de ejecutar, estudiar, modificar y distribuir el software. Estableció la Free Software Foundation (FSF) para respaldar el proyecto GNU..

Linus Torvalds

En 1991, Linus Torvalds, un estudiante finlandés, creó el núcleo Linux como parte del proyecto GNU.

El núcleo Linux:

    Linux proporcionó el componente que faltaba para tener un sistema operativo completo.
  • Sistema operativo completo:
    Distribuciones Linux- A partir de ahí, se combinó el núcleo Linux con herramientas y software del proyecto GNU, así como de otras fuentes, para formar distribuciones Linux completas y funcionales.
  • Expansión y Diversificación:
    • Crecimiento: Linux creció rápidamente debido a su naturaleza de código abierto, atrayendo contribuciones de desarrolladores de todo el mundo.
    • Diversificación: Se desarrollaron numerosas distribuciones Linux para satisfacer diferentes necesidades y preferencias, como Ubuntu, Fedora y Debian.
  • Actualidad:
    • Uso: Hoy en día, Linux se utiliza en una variedad de dispositivos, desde servidores hasta teléfonos inteligentes (Android) y sistemas embebidos.
    • Comunidad Activa: La comunidad de desarrollo de Linux sigue siendo activa, con contribuciones continuas y actualizaciones constantes.
  • En resumen, Linux es el resultado de la combinación del núcleo desarrollado por Linus Torvalds con las herramientas y el software del proyecto GNU, y ha evolucionado para convertirse en un sistema operativo ampliamente utilizado en diversas aplicaciones.
